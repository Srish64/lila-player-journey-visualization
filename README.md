# Lila Player Journey Visualization

## Project Overview
The Lila Player Journey Visualization project aims to provide an in-depth analysis and visualization of player journeys within the Lila platform. By utilizing various data visualization techniques, this project enables stakeholders to understand player engagement, progression, and retention.

## Features
- Interactive visualizations of player journeys
- Detailed insights into player engagement metrics
- Ability to filter data based on various parameters
- Export capabilities for visualizations and data reports
- User-friendly interface for exploring player statistics

## Technical Stack
- **Frontend:** React.js, D3.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Data Processing:** Python, Pandas
- **Deployment:** Docker, AWS

## Data Visualization Methodology
The project employs a range of data visualization methods including:
- Time series analysis to track player activities over time.
- Heat maps to represent engagement levels across different player segments.
- Network graphs to visualize player relationships and interactions.

## Usage Guide
To run this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Srish64/lila-player-journey-visualization.git
   ```
2. Navigate to the project directory:
   ```bash
   cd lila-player-journey-visualization
   ```
3. Install dependencies for the backend:
   ```bash
   cd backend
   npm install
   ```
4. Start the backend server:
   ```bash
   npm start
   ```
5. Navigate to the frontend directory and install dependencies:
   ```bash
   cd ../frontend
   npm install
   ```
6. Start the frontend application:
   ```bash
   npm start
   ```
