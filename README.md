# LILA BLACK – Player Journey Visualization Tool

## Live Demo
https://srish64.github.io/lila-player-journey-visualization/

## Overview
This tool enables Level Designers to visualize player movement, combat hotspots, storm deaths, and map usage patterns across 5 days of production gameplay data.

## Features
- Map filtering (AmbroseValley, GrandRift, Lockdown)
- Match filtering
- Bot vs Human distinction
- Heatmap overlays
- Match playback
- Zero-dependency, self-contained HTML

## How to Run
Open `lila_viz_tool.html` in any modern browser (Chrome recommended).
